Cont.ext.

O que é Cont.ext.?

Em diversas áreas de engenharia, matemática, física, computação e análise de dados é necessário a extração de dados como o contorno, a área ou até mesmo o perimetro de um objeto a partir de uma imagem. As soluções disponíveis atualmente permitem a realização desse processo de maneira semi-automatizada, selecionando manualmente o contorno desejado. Nós percebemos que esse processo poderia ser ainda mais intuitivo, fácil e parametrizável, poupando tempo e esforço. Por conta disso, desenvolvemos Cont.ext., uma solução que permite a extração dos dados necessários a partir de algoritmos de processamento de imagens.


Filtro HSV

GUI (Em desenvolvimento) para a binarização de imagens a partir de limiares HSV utilizando da biblioteca OpenCv. Dessa forma é possível isolar e exportar o objeto que deseja na imagem de forma automatizada em formato PNG. Com essa ferramenta voce possui uma prévia do resultado final durante o procedimento e não precisa selecionar manualmente os pontos.

Extração de Contorno

CLI para a extração de contornos e metadados de imagens binárias utilizando do algoritmo Moore's Neighborhood, permitindo a exportação e importação em outras ferramentas de maneira facilitada, como o Matlab e o Octave. Nosso CLI também conta com a parametrização de intervalos, escala e deslocamento.

Construção de Malhas

CLI para a construção das malhas interiores ao objeto selecionado, permitindo a seleção e parametrização dos pontos internos ao contorno e também do nível de refinamento a partir de retangulos. A CLI também conta com a exportação facilitada dos dados e metadados para outras ferramentas, como Matlab e Octave.
